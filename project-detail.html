Background

Refrigeration system #2 serves a freezer with 1,000,000 sq ft and has 1,000 evaporators that are identical with 4fans x 3 hp each. The control system has the ability to cycle the fans with the liquid solenoid valve, however, this function is not enabled and the fans run constantly except during a defrost cycle.

This saves energy by reducing fan runtime as well as reduces refrigeration load when the system is not calling for cooling.

The solenoid cycle time is 70% on average for the facility, and it is anticipate the off cycle time is 30%.

Required Actions

Enable fan cycling on each fan evaporator control screen, and set "stir cycle" on each evaporator to run the evaporator fans a minimum of 5 minutes per hour to ensure stagnant air is mitigated.

Monitor room air temperatures in low air movement areas of the freezer and increase stir cycle time by 5 minutes until issues are mitigated.
